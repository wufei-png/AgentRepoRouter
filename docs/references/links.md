# 参考资源

## 核心项目

- [OpenClaw](https://github.com/openclaw/openclaw) - AI agent 控制平台
- [ACPX](https://github.com/openclaw/acpx) - ACP 协议实现
- [MCO](https://github.com/mco-org/mco) - Multi-agent orchestrator
- [A2A](https://github.com/a2aproject/A2A) - Agent-to-Agent 协议

## 协议规范

- [ACP (Agent Client Protocol)](https://docs.openclaw.ai/cli/acp) - IDE ↔ Agent 协议
- [MCP (Model Context Protocol)](https://modelcontextprotocol.io) - Agent ↔ Tools 协议
- [A2A Protocol](https://a2a-protocol.org) - Agent ↔ Agent 协议
